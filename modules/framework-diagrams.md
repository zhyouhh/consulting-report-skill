# 战略框架图制作模块

## 一、框架图的作用

框架图用于可视化展示战略分析、业务逻辑、组织关系等，提升报告的专业性和可读性。

## 二、常用框架图类型

### 2.1 SWOT 分析图
```
优势（Strengths）        机会（Opportunities）
- 优势1                  - 机会1
- 优势2                  - 机会2

劣势（Weaknesses）       威胁（Threats）
- 劣势1                  - 威胁1
- 劣势2                  - 威胁2
```

### 2.2 Porter 五力模型
```
            潜在进入者
                ↓
供应商 → 现有竞争者 ← 买方
                ↑
            替代品
```

### 2.3 价值链分析
```
支持活动：基础设施、人力资源、技术开发、采购
主要活动：进货物流 → 生产 → 出货物流 → 营销 → 服务
```

### 2.4 BCG 矩阵
```
市场增长率
高 |  问题  |  明星  |
   |--------|--------|
低 |  瘦狗  | 现金牛 |
    低      高
    相对市场份额
```

## 三、制作工具

### 3.1 Python + Matplotlib
适合数据驱动的图表

### 3.2 Mermaid
适合流程图、组织图

### 3.3 PowerPoint
适合复杂的战略框架图

## 四、设计原则

1. **简洁明了**：避免过度装饰
2. **逻辑清晰**：关系明确
3. **配色专业**：使用统一配色
4. **标注完整**：必要的说明文字
