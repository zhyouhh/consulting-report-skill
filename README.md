# 咨询报告写作助手

> 📊 将咨询报告写作从一次性对话，升级为可追踪、可恢复、可复用的工程化协作流程。

一个面向咨询顾问、商业分析师、战略规划人员的 Claude Skill，提供完整的报告协作系统。

## 核心功能

- **全流程协作** - 从项目启动到演示准备，按阶段门禁执行
- **专业化写作** - 金字塔原理、MECE、去AI化规范
- **商业分析框架** - SWOT、Porter 五力、PEST、价值链等
- **商业图表生成** - 瀑布图、热力图、漏斗图、甘特图、桑基图
- **计划可恢复** - `plan/` 持续记录目标、产出、决策

## 安装

```bash
# 复制 skill 目录到你的项目
cp -r consulting-report-skill /path/to/your/project/
```

## 快速开始

```
# 在支持的 IDE 中加载 skill
/use consulting-report-skill

# 开始写作
"帮我写一份市场进入战略报告"
```
