# 咨询报告模板库

## 一、模板使用指南

本模块提供 4 种常用咨询报告模板，每种模板包含完整的章节结构和填写指南。

## 二、战略咨询报告模板

### 2.1 标准结构

```markdown
# [公司名称]战略咨询报告

## 执行摘要
[1-2 页，包含背景、关键发现、核心建议、下一步行动]

## 1. 项目背景与目标
- 客户背景
- 核心问题
- 研究范围
- 项目目标

## 2. 外部环境分析
### 2.1 行业趋势（PEST 分析）
- 政治因素
- 经济因素
- 社会因素
- 技术因素

### 2.2 竞争格局（Porter 五力）
- 现有竞争者
- 潜在进入者
- 替代品威胁
- 买方议价能力
- 供应商议价能力

## 3. 内部能力分析
### 3.1 SWOT 分析
- 优势（Strengths）
- 劣势（Weaknesses）
- 机会（Opportunities）
- 威胁（Threats）

### 3.2 价值链分析
- 主要活动
- 支持活动
- 成本结构

## 4. 战略选项
### 4.1 战略方向
- 增长战略
- 竞争战略
- 差异化策略

### 4.2 方案对比
| 方案 | 优势 | 劣势 | 风险 | 预期收益 |
|------|------|------|------|----------|

## 5. 推荐方案
### 5.1 核心战略
### 5.2 关键举措
### 5.3 资源需求

## 6. 实施路线图
- 短期（0-6 月）
- 中期（6-18 月）
- 长期（18-36 月）

## 7. 风险与缓解措施

## 附录
- 数据来源
- 详细分析
- 补充图表
```

## 六、快速报告生成（模板化交付）

### 6.1 适用场景

当遇到以下情况时，使用快速报告生成模式：
- 合同要求但客户不会细看的配套文档
- 主要为满足形式要求、凑够篇幅的报告
- 时间紧迫，需要快速产出的标准化报告
- 基于公开信息的行业背景、政策梳理等

### 6.2 快速生成原则

**执行前必须确认**：
- 目标篇幅（多少页？）- 这是核心要求
- 交付时间（什么时候要？）
- 基本要求（合同/客户有特殊格式要求吗？）

**生成原则**：
1. **使用标准框架** - 采用行业通用的分析框架，无需定制化
2. **公开信息为主** - 基于行业报告、政策文件等公开资料
3. **确保结构完整** - 章节齐全、逻辑清晰、格式规范
4. **精确控制篇幅** - 达到目标页数（±1页），不多不少
5. **看起来专业** - 图表齐全、排版美观、数据标注规范

### 6.3 常见快速报告类型

#### 6.3.1 行业背景报告（10-15页）

```markdown
# [行业名称]行业背景报告

## 1. 行业概述（2-3页）
- 行业定义与范围
- 发展历程（简要时间线）
- 产业链结构图

## 2. 市场规模（2-3页）
- 当前市场规模（引用权威报告）
- 历史增长趋势（近3-5年数据）
- 未来预测（引用行业预测）

## 3. 政策环境（2-3页）
- 主要政策法规列表
- 政策影响分析（简要）
- 政策趋势

## 4. 竞争格局（2-3页）
- 主要参与者列表
- 市场份额分布（饼图/柱状图）
- 竞争特点（3-5点）

## 5. 发展趋势（2-3页）
- 技术趋势（3-5点）
- 市场趋势（3-5点）
- 未来展望

## 附录
- 数据来源列表
```

#### 6.3.2 政策法规梳理（8-10页）

```markdown
# [领域]政策法规梳理报告

## 1. 政策体系概述（1-2页）
- 政策层级（国家-地方）
- 主管部门
- 政策框架图

## 2. 国家层面政策（3-4页）
| 政策名称 | 发布时间 | 主管部门 | 核心内容 |
|---------|---------|---------|---------|

## 3. 地方层面政策（2-3页）
按省份/城市分类列举

## 4. 政策影响分析（1-2页）
- 对行业的影响
- 对企业的影响
- 合规要求

## 5. 政策趋势（1页）
```

#### 6.3.3 标准分析框架报告（8-10页）

**SWOT 分析报告**：
```markdown
# [主题] SWOT 分析报告

## 1. 分析背景（1页）

## 2. SWOT 分析（4-5页）
### 2.1 优势（Strengths）
- 优势1：描述 + 数据支撑
- 优势2：描述 + 数据支撑
- 优势3：描述 + 数据支撑

### 2.2 劣势（Weaknesses）
- 劣势1：描述 + 影响分析
- 劣势2：描述 + 影响分析

### 2.3 机会（Opportunities）
- 机会1：市场机会 + 规模估算
- 机会2：政策机会 + 时间窗口

### 2.4 威胁（Threats）
- 威胁1：竞争威胁 + 应对建议
- 威胁2：市场威胁 + 风险评估

## 3. SWOT 矩阵（1页）
四象限矩阵图

## 4. 战略建议（2-3页）
- SO 策略（优势+机会）
- WO 策略（劣势+机会）
- ST 策略（优势+威胁）
- WT 策略（劣势+威胁）
```

### 6.4 快速生成技巧

1. **引用权威数据** - 使用行业报告、政府统计数据，标注来源
2. **使用通用图表** - 饼图、柱状图、趋势线等标准图表
3. **列表式呈现** - 多用表格和列表，减少长段落
4. **模块化填充** - 每个章节独立，可快速替换内容
5. **控制深度** - 点到为止，不做深入分析

### 6.5 质量底线

即使是快速生成的报告，也要确保：
- ✅ 数据来源标注清晰
- ✅ 图表编号连续
- ✅ 格式规范统一
- ✅ 无明显错误
- ✅ 逻辑基本通顺

## 七、模板使用建议

1. **选择合适模板**: 根据项目类型选择对应模板
2. **调整章节结构**: 根据实际需求增删章节
3. **保持逻辑一致**: 确保各章节逻辑连贯
4. **数据驱动**: 用数据支撑每个结论
5. **客户导向**: 关注客户最关心的问题
